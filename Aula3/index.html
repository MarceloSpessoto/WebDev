O que aprendi

  • Aula 1 - Introdução ao HTML e desenvolvimento Web:

    Foram apresentados os seguintes tópicos:

    1. Conceito de desenvolvimento Web:
      • Web x Internet;
      • Linguagens(não necessariamente de programação) relacionadas ao desenvolvimento web: HTML, CSS, JS
    2. Introdução ao HTML:
      • Características básicas de HTML
      • Estruturação de código HTML e analogia com árvores
      • Apresentação de tags fundamentais
      • Atributos de tags e tags autocontidas
    3. Introdução a conceitos básicos de git.

  • Aula 2 - CSS:

    Foram apresentados os seguintes tópicos:

    1. Introdução ao CSS:
      • "Cascading Style Sheet"
      • Uso para estilização
      • Conceito de Herança
      • Uso de seletores para estilizar página:
        1. Tag (Menor prioridade)
        2. Classe
        3. ID (Maior prioridade)
      • Tipos de estilização:
        • Interna (Usada para estilizar essa página)
        • Externa
      • Box Model ("Caixas que compõem a página")
        • Caixas compostas por:
          • Conteúdo
          • Preenchimento (Padding)
          • Borda
          • Margem
        • Unidades de medida:
          • pixels (px)
          • largura do char "m" na fonte atual (em)
          • largura do char "m" na fonte original (rem)
          • 1/100 da altura e largura do viewport (vh e vw)
          • medida relativa (%)
