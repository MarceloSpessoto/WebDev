Aula 2 - CSS:

Foram apresentados os seguintes tópicos:

  1. Introdução ao CSS:
    • "Cascading Style Sheet"
    • Uso para estilização
    • Conceito de Herança
    • Uso de seletores para estilizar página:
      1. Tag (Menor prioridade)
      2. Classe
      3. ID (Maior prioridade)
    • Tipos de estilização:
      • Interna (Usada para estilizar essa página)
      • Externa
    • Box Model ("Caixas que compõem a página")
      • Caixas compostas por:
        • Conteúdo
        • Preenchimento (Padding)
        • Borda
        • Margem
      • Unidades de medida:
        • pixels (px)
        • largura do char "m" na fonte atual (em)
        • largura do char "m" na fonte original (rem)
        • 1/100 da altura e largura do viewport (vh e vw)
        • medida relativa (%)
